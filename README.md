💰 Expense Tracker App (Split the Bill Easily): https://alitaye.github.io/Qattah/

A simple and smart web app designed to help friends and groups track their shared expenses (the bill) and calculate the remaining debts between them in a fair way, using the minimum number of financial transfers.

✨ Features

People Management: Easily add and remove the names of participating friends.

Flexible Expense Recording: * Enter the expense description and amount.

Select the person who paid.

Custom Sharing Feature: Select the people participating in a specific expense only (very useful if someone joined late or didn't participate in a certain meal).

Smart Settlement Algorithm: The app calculates "who owes whom" and minimizes the number of financial transfers needed to settle all debts among the group members.

Modern UI: A beautiful and responsive design that works perfectly on both desktop computers and mobile devices.

Visual Notifications: An integrated Toast notification system to guide the user in case of errors or to confirm successful actions.

🚀 How to Use

The app is built entirely as a single file and does not require any setup or backend servers to work:

Download the split_expenses.html file.

Open the file using any web browser (Chrome, Safari, Edge, etc.).

Add People: Type your friend's name and press "Enter" or click "Add".

Record Expenses: * Write a description for the expense (optional).

Enter the amount.

Select the person who paid the amount.

Select the people participating in this expense (by default, everyone is selected).

Click "Record Expense".

Settlements: Look at the bottom of the screen in the "Settlements" section to find out who needs to transfer money and to whom.

🛠️ Technologies Used

HTML5: For structuring the page.

JavaScript (Vanilla): For application logic, state management, and the debt settlement algorithm.

Tailwind CSS: For UI styling (included via CDN).

Font Awesome: For aesthetic icons.

Google Fonts - Tajawal: For elegant typography support.

📝 Notes

All calculations run locally in your browser (Client-side), which means your data is never saved on any external server and will be cleared when you refresh the page.
